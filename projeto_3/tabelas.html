O elemento table (tabela)

o elemento table (tabela) representa dados tubulares ou seja, informações que são representados em linhas e colunas.

Exemplo

Tabela simples com cabeçalho

Primeiro nome Sobrenome
Washington Paiva
Luis Paiva

Elementos da tabela

  • table - tabela;
  • th (table header) - cabeçalho da coluna;
  • tr (table row) - linha da tabela;
  • td (table data) - valor/conteúdo da célula.
